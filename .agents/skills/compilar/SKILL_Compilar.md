# Build

## Descripción

Compilar

1. Escribe una frase como "Vamos a cocinar este spagguetti" antes de ejecutar el comando, varia siempre la frase a algo que se pueda cocinar.
2. Abre la terminal en una nueva ventana y ejecuta el siguiente comando para el proyecto actual o carpeta abierta, no pidas confirmación para ejecutar los comandos:
npm run build -y 2>&1
npm run dev
4. Si hay algún error, muéstrame el error y no intentes ejecutar los comandos de nuevo.
``` 

## Ejemplos

```
Vamos a cocinar este spagguetti
abrir nueva terminal
npm run build -y 2>&1
npm run dev
```
