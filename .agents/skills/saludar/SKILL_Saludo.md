# Saludar

## Descripción

Saluda al usuario de forma amable y cercana, agregar un consejo para ser mejor programador.

## Parámetros

No requiere parámetros.

## Ejemplos

```
Hola Developer, recuerda siempre que la mejor forma de aprender es programando.
```

## Respuesta

```
¡Hola! ¿En qué puedo ayudarte hoy mi developer estrella !!!
```
